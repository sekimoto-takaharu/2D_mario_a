# Phaser 2D Platformer Demo

Phaser を使って作成した、シンプルな **2D 横スクロールアクションゲーム** です。  
ゲーム内でコース選択ができ、テーマの異なる複数のコースをプレイできます。

---

## 🎮 ゲーム概要

- ゲーム内コース選択画面あり
- 横スクロール型 2D アクション
- コースごとに操作感・重力・ギミックが変化

### 実装済みコース

| コース | 内容 |
|------|------|
| 地上コース | 基本的な操作感の入門ステージ |
| 地下コース | 暗い雰囲気のステージ |
| 水中コース | 浮上・潜水ができる水中挙動 |
| 城コース | マグマに触れると即ゲームオーバー |

---

## 🕹 操作方法

| キー | 動作 |
|----|----|
| ← / → | 移動 |
| ↑ | ジャンプ（地上） / 浮上（水中） |
| ↓ | 潜水（水中） |
| ESC | コース選択画面へ戻る |

## 📁 ディレクトリ構成

```text
src/
  main.js                # Phaser 初期化
  style.css              # 画面全体のスタイル
  game/
    courses.js           # コース定義（データ駆動）
    utils/
      textures.js        # 四角形テクスチャ生成
    scenes/
      MenuScene.js       # コース選択画面
      GameScene.js       # ゲーム本編
      ClearScene.js      # クリア画面
```

---

## 🧠 設計方針

- **Scene は共通化**し、コースごとの差分は `courses.js` で管理
- コースごとに以下の要素を切り替え
  - 背景色
  - 重力
  - プレイヤー操作感（加速・最高速・ジャンプ）
  - 足場配置
  - ギミック（水中 / マグマ）
- 画像素材を使用せず、すべてコードで描画
  - Phaser の `Graphics` を利用してテクスチャを生成
